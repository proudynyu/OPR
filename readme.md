# Odin Pdf Reader

Reading a PDF using the Odin Language Program.

This is a study case to learn how the PDF file formats works and how to handle this kind of format using Odin.

## Build

```shell
-> odin build .

-> ./pdf_reader
```

Can run directly using

```shell
-> odin run .
```
